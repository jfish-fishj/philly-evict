\section{Overview}\label{overview}

This writeup isolates the maintenance-complaint relationship using
\textbf{severe complaints only}, before returning to demographic
regressions.

The underlying script is \texttt{r/severe-maintenance-diagnostics.R}.

Design choices in this version:

\begin{itemize}
\tightlist
\item
  Outcome: \texttt{total\_severe\_complaints} (quarterly count)
\item
  Panel + controls source: quarterly rental panel
  (\texttt{building\_data\_rental\_quarter}) with annual controls merged
  from \texttt{bldg\_panel\_blp} (\texttt{PID\ x\ year})
\item
  Maintenance flow baseline: repair-ish permit counts only
  (\texttt{electrical}, \texttt{plumbing}, \texttt{mechanical},
  \texttt{fire\_suppression}); a broad flow variant adds
  \texttt{building\_permit\_count} for robustness
\item
  Maintenance scaling: per-unit rates using annual \texttt{total\_units}
  (\texttt{maint\_rate\_*\ =\ permit\_flow\ /\ total\_units\_ann}, with
  \texttt{total\_units\_ann} floored at 1)
\item
  Lag/lead maintenance variables: \texttt{t-1} flow, \texttt{t-1:t-4}
  average lagged flow, and leads (\texttt{t+1}, \texttt{t+2}) for
  falsification
\item
  Rolling maintenance stock: EWMA of \textbf{lagged} repair flow
  (\texttt{maint\_stock\_lag}) within \texttt{PID}, then scaled by units
  (\texttt{maint\_stock\_lag\_rate}); default decay parameter is
  \texttt{lambda\ =\ 0.9} (CLI-overridable)
\item
  Unit controls: nonlinear spline in \texttt{log(1\ +\ total\_units)}
  (\texttt{df\ =\ 4}) plus deterministic fine unit bins (\texttt{1},
  \texttt{2}, \ldots, \texttt{10}, \texttt{11-12}, \texttt{13-15},
  \ldots, \texttt{151-200}, \texttt{201+}, \texttt{Unknown}) used in the
  stock-focused specifications
\item
  Quality controls: coarsened \texttt{A/B/C/D/E/Other} grades (dropping
  \texttt{+/-}) throughout the diagnostics
\item
  Estimation: \texttt{fepois} with cluster-robust SEs by \texttt{PID},
  weighted by annual units
\item
  Diagnostics added: lead falsification, PID+period FE (within-building)
  regressions, support tables for quality x fine-unit-bin cells
\item
  Parcel-controls diagnostics: the default fast-mode stock table
  includes a parcel-controls stock spec with \texttt{log(total\_area)}
  (no market-value control)
\item
  Runtime note: \texttt{r/severe-maintenance-diagnostics.R} defaults to
  \texttt{-\/-fast\_mode=TRUE}, which fits and writes tables in batches,
  skips the slowest stock-quality interaction variants /
  rich-interaction or rent variants / PID FE variants, and omits the
  partial-correlation figure; use \texttt{-\/-fast\_mode=FALSE} for the
  full suite
\end{itemize}

\section{Notes}\label{notes}

\begin{itemize}
\tightlist
\item
  Category-specific complaint counts exist in the quarterly product, but
  category-specific \textbf{severe} complaint counts are not available
  in the current product, so this writeup remains severe-total only.
\item
  Stock-by-quality interaction specs are estimated on the
  controls-matched subset with non-\texttt{Unknown} fine unit bins, to
  avoid collapsing identification onto the missing-controls mass.
\item
  This is a diagnostics workflow intended to decide which maintenance
  specifications are worth carrying into later complaint or demographic
  analyses.
\end{itemize}
